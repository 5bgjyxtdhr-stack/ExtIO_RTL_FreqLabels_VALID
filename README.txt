# ExtIO_RTL_FreqLabels (VALID skeleton)

- 32-bit (x86) ExtIO DLL skeleton for HDSDR/Winrad interface.
- Exports provided via `.def` to ensure undecorated names HDSDR expects.
- No GUI/User32 dependency.

## Build
Use GitHub Actions workflow in `.github/workflows/build.yml` or run `build.bat` from **VS Developer Command Prompt for x86**.
